# php box

A wercker box based off the official wercker NodeJs and PHP boxes to combine them both into one

Installed NodeJs version:

- Nodejs `0.10.33`

Installed PHP version:

- PHP `5.5.20`

Installed PHP extensions:

- bz2
- intl
- calendar
